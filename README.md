# **CaveSnagger** 

**CaveSnagger** is a streamlined research acquisition tool built under the **CavemanAI** brand. It is designed to bridge the gap between complex academic repositories and everyday users by providing a simple, high-performance interface for "snagging" research papers from the arXiv open-access archive.

The application follows the **CavemanAI** philosophy: high-intelligence capability delivered through an interface that is accessible and easy to use for everyone.

### **Features**

* **Targeted Acquisition**: Search by any topic of interest, from Theoretical Physics to Machine Learning.  
* **SafeCoffee Palette**: Implements the signature branding style with deep navy backgrounds, teal accents, and purple highlights.  
* **Automated Filing**: Automatically cleans and formats paper titles for local storage in your designated "Acquisitions" folder.

### **Technical Backbone**

* **Language**: Python 3.12  
* **GUI Framework**: PySide6  
* **Web Scouring**: BeautifulSoup4 and Requests  
* **Data Source**: arXiv API

### **Installation**

1. Ensure you have Python 3.12+ installed.  
2. Install the required dependencies:
   
    bash
    pip install -r requirements.txt  
    
4. Place your branding asset (`Snagger.jpeg`) in the same directory as the script.  
5. Launch the application:
   
   Bash  
   python CaveSnagger.py
